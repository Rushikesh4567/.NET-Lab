# 🧪 C# / .NET Lab Experiments Repository

## 📌 Overview

This repository contains all my **C# and .NET lab experiments** performed as part of my coursework.

Each experiment includes:

* 📖 Objective
* 🧠 Theory
* 💻 Implementation (Code)
* 📊 Output
* ✅ Conclusion

This repository serves as a **complete practical reference** for C# and ASP.NET Core concepts.

    * .NET Desktop Development
    * ASP.NET and Web Development

* ⚙️ .NET 8 SDK

* 🌐 Postman (for API testing)

* 🗄️ SQL Server Express *(optional for EF Core)*
